# Welcome to rBundle
The go-to Roblox package manager with no external dependencies.

## Why rBundle?
rBundle provies an extensive feature set alongside primitive version control, GitHub support, and dynamic bundles[^1]. Moreover, many other so called package managers for Roblox have their packages hardcoded. They take a *static* approach and limit user control. rBundle takes a *dynamic* approach keeping developers developing and not having to worry about if their bundles are up to date.

[^1]: A bundle is rBundle terminology for a package
